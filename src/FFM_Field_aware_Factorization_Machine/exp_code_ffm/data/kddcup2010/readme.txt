Usage:
1. Download compressed "kddb-raw" file at "https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/binary/kddb-raw.bz2"
   and compressed "kddb-raw.t" at "https://www.csie.ntu.edu.tw/~cjlin/libsvmtools/datasets/binary/kddb-raw.t.bz2". Then,
   uncompress them in this directory.
2. Execute "sh run.sh"
